# OMRON_Blood_Pressure_Analyse
欧姆龙血压计数据分析（csv）

## 目的
项目主要用于学习 tailwind css 框架，具体技术栈：
1. vue3.0 + ts
2. tailwind
3. echarts 
4. papaparse(读取转换 csv)
